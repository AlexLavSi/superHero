# superHero
    for start project:
        1. install docker and doker-compose
        2. run the command: docker-compose up --build -d 
        3. open tab browser to address 'http://localhost:4200/'
        
    for run tests:
        1. install docker and doker-compose
        2. run the command: docker-compose up --build -d
        3. go to folder `backEnd`
        4. run the command: npm test
